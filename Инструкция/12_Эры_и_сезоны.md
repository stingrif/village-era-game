# Эры и сезоны

## Структура

- **1 эра = 7 дней**
- **Сезон = 4 эры = 28 дней**
- В конце 4-й эры — выдача призов

## Перенос между эрами (1→2→3→4)

**Переносится:**
- здания и уровни
- инвентарь (реликвии, амулеты, яйца, буквы)
- звёзды, опыт
- ресурсы (wood, stone)
- монеты

**Сбрасывается:**
- активные порчи/бафы

## Перенос между сезонами (после 4-й эры)

| Ресурс | Перенос |
|--------|---------|
| Монеты | 25% |
| Ресурсы | 30% |
| Здания | сохраняются |
| Рейтинг/медали | архивируются |

```
coins_next = floor(coins_current * 0.25)
wood_next  = floor(wood_current * 0.30)
stone_next = floor(stone_current * 0.30)
```

## Очки эры (era_points)

- Считаются **отдельно** от монет
- **Не выводятся**
- Начисление: улучшения зданий, сжигание реликвий, редкости, вклад в профиль
- При вкладе: `era_points += diamonds * 10` (по withdraw_credit_rules)

## Лидерборды

- **weekly_points**, **monthly_points**, **era_points**
- Призы: из `payout_pool` (TON) и/или внутриигровые
- Таблица `leaderboards` (period, period_key, user_id, points)

### Тайбрейки (при равных era_points)

Для era-лидерборда при равных era_points первый тайбрейк (больше era_points) не применяется. Используются:
1. Меньше `risk_score`
2. Раньше достиг порога (timestamp)
