| Здание | Механика |

| Дома | Главный доход монет/час; слоты для fire_01; пассив skyfall_loss_reduction |
| Ферма | Стабильный доход; слоты для fire_02, yin_05/06; пассив curse_resist |
| Рынок | Торговля: снижает налог, повышает цену продажи; fire_03/05, yin_07, yan_05 |
| Шахтный офис | Попытки шахты (+попытки/сутки), бонус к добыче; открывает доступ к шахте |
| Кузница | PvP-порчи: усиливает силу и резист; yan_01/02/03/04/06/07; доп. YAN из Deep Pool — [02](02_Реликвии_и_эффекты.md) |
| Храм | Пассив remove_one_curse (1 раз/эру); carryover; открывает эффекты cleanse из Deep Pool (YIN deep) — [02](02_Реликвии_и_эффекты.md) |

| Памятник эры, Портал | Очки эры; yan_08 |

## Зависимости крафта

### Merge (слияние)

- **Требует:** 3 реликвии одного типа (fire/yin/yan) + dust
- **Стоимость:** `dust = 40 * avg_level`
- **Результат:** новый предмет из deep pool
- **Зависит от:** инвентарь реликвий, dust

### Upgrade (усиление)

- **Требует:** 1 предмет + dust
- **Стоимость:** `dust = 60 * current_level`
- **Риск:** поломка 15% (10% после рекламы)
- **Зависит от:** уровень предмета

### Reroll (перекат эффекта)

- **Требует:** 1 предмет + dust
- **Стоимость:** `dust = 25 * level`
- **Риск:** поломка 15%
- **Зависит от:** уровень предмета

Две схемы: Dust или COINS+STARS. Подробнее: [05_Крафт_и_слияние.md](05_Крафт_и_слияние.md).

## Зависимости вывода TON

Все условия должны быть выполнены (уровень ≥ 3, привязан кошелёк, не restricted, правило 10%). Зачёт в eligibility и детали — в [08_Вывод_и_антихалява.md](08_Вывод_и_антихалява.md).

## Цепочки прогрессии

```
Чекин → Попытки → Шахта → Лут (реликвии/монеты/яйца) → Инвентарь
                                                              ↓
Здания → Слоты реликвий → Реликвии → Усиление дохода     Крафт → Merge/Upgrade/Reroll
                                                              ↓
                                                         Рынок (продажа/обмен)
```

- **Чекин** — источник попыток шахты (10ч CD, 3 попытки).
- **Шахта** — источник лута (реликвии, амулеты, монеты, яйца).
- **Здания** — дают доход, слоты для реликвий, пассивы (Defense Tier, skyfall, curse_resist).
- **Крафт** — превращает лут в более сильные предметы или dust.

Подробнее: [03_Шахта_и_яйца.md](03_Шахта_и_яйца.md), [05_Крафт_и_слияние.md](05_Крафт_и_слияние.md), [11_Рынок_и_P2P.md](11_Рынок_и_P2P.md).

## Defense Tier: зависимости

Защитные здания дают Defense Power; от него зависят Tier 2/3 и снятие strong/severe. Веса зданий, формула def_power и правило перекрытия сильной порчи — в [04_Бафы_и_порчи.md](04_Бафы_и_порчи.md).
