# Крафт и слияние

## Две схемы (на выбор реализации)

### Схема A: Dust (Пепел)

- **Валюта:** Dust — из разбора реликвий/лута.
- **Merge (3→1):** 3 реликвии одного типа → `dust_cost = 40 * avg_level`. Результат: 70% avg_level, 25% +1, 5% +2 (кап 5). Эффект из deep pool.
- **Upgrade:** 1 предмет, `dust_cost = 60 * level`. Успех 65%, без изменений 20%, поломка 15%. Реклама: поломка → 10%.
- **Reroll:** `dust_cost = 25 * level`. Новый эффект 85%, поломка 15%.

### Схема B: COINS + STARS

| Операция | Стоимость | Шансы |
|----------|-----------|--------|
| Merge-2 | 600 coins + 1200 stars | 72% обычный, 18% выше, 10% слом |
| Merge-3 | 900 coins + 1800 stars | 80% обычный, 17% выше, 3% слом |
| Upgrade | 1200 coins + 2500 stars | 75% +1, 20% без изменений, 5% слом |
| Break | 150 coins | шанс ресурсов/звёзд |

## Уровни предмета (1..3)

- lvl2: сила эффекта ×1.15  
- lvl3: ×1.30  

## Реклама («Усилить попытку»)

- Merge-2: слом 10% → 8%  
- Merge-3: +1% к успеху  
- Upgrade: +2% к успеху  
- **Лимит:** 3 усиления/сутки.

## ad_token (10 мин после рекламы)

- Merge: шанс +2 level 5% → 8%  
- Upgrade: поломка 15% → 10%  
- Reroll: поломка 15% → 12%  

## Слияние 2/3 реликвии → Magic

| Условие | Результат |
|---------|------------|
| 2 реликвии | Magic |
| 3 реликвии | Magic (усиленный) |
| Эффект | ~×1.5 от среднего |
| Риск | ↑ потери при Падении неба |
