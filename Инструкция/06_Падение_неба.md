# Падение неба (Skyfall)

## Когда

- **Суббота** — режим «Падение неба»: доход не начисляется, монеты убывают по формуле.
- Минимальный съём всегда; нельзя «переждать» без потерь.

## Компоненты потерь

1. **От мощности деревни** (income)  
2. **От баланса** (ломает стратегию «ничего не строю»)

## Константы

| Параметр | Значение |
|----------|----------|
| sf_k_power | 0.18 (18% от hourly income за час) |
| sf_k_balance | 0.006 (0.6% от баланса за час) |
| sf_min_loss_per_tick | 12 |
| sf_max_loss_per_tick | 3500 |
| **R_balance (кап снижения)** | **макс 35%** суммарно |

## Формула за 1 тик (1 час)

```
loss_power   = I * sf_k_power * R_power
loss_balance = B * sf_k_balance * R_balance   // R_balance с капом 35%
loss_tick = clamp(max(sf_min_loss_per_tick, floor(loss_power + loss_balance)), 0, sf_max_loss_per_tick)
balance_coins = max(0, balance_coins - loss_tick)
```

I = income_per_hour_total, B = balance_coins, R_power / R_balance — от реликвий и зданий.

## Штраф «пустой деревни»

При **< 3 построек** на поле:

- sf_k_balance × 1.6  
- sf_min_loss_per_tick × 1.5  

## Влияние на здания

| Фактор | Эффект |
|--------|--------|
| Высокий доход | ↑ потери |
| Сильные реликвии | ↑ риск |
| Порча | ослабление эффектов |
| Низкая активность | доп. штраф |
