# Реликвии и эффекты

Геймплейные эффекты и ограничения реликвий. Визуал и анимации см. [19_Список_предметов.md](19_Список_предметов.md).

---

## Типы реликвий и базовые эффекты

| Редкость | Тип эффекта | Базовое действие |
|----------|-------------|------------------|
| Fire (Огонь) | Экономика | +доход / +продажа |
| Yin (Инь) | Защита | −потери / −порча |
| Yan (Янь) | Агрессия | +порча / −доход врагу |
| Tsy (Цы) | Тактика | заморозка / перераспределение |
| Magic | Усиление | комбинированный эффект |
| Epic | Уникальный | особый эффект (1 на игрока) |

## Ограничения установки

| Правило | Значение |
|---------|----------|
| Одинаковые реликвии | нельзя в одном здании |
| Стак эффектов | запрещён |
| Активных реликвий одного типа | ≤ 2 на деревню |
| Epic реликвии | 1 на игрока |

## Relic Core vs Relic Artifact

1. **Relic Core (пассив)** — ставится в слоты зданий, работает постоянно.
2. **Relic Artifact (активируемый)** — одноразовый баф: активировал → работает N минут → сгорает.

Подробнее: [04_Бафы_и_порчи.md](04_Бафы_и_порчи.md).
