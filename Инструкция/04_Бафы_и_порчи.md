# Бафы и порчи

## Бафы (self)

| Качество | Эффект | Длительность |
|----------|--------|--------------|
| Обычный | слабый бонус | 6 ч |
| Редкий | средний бонус | 12 ч |
| Магический | сильный бонус | 24 ч |

**Ограничения:** 1 баф активен, не даёт иммунитета.

## Порчи (PvP)

| Качество | Эффект | Длительность |
|----------|--------|--------------|
| Обычная | +потери / −доход | 6 ч |
| Редкая | +потери + блок | 12 ч |
| Магическая | комбинированная | 24 ч |

**Ограничения:** 1 порча на цель, cooldown на повтор, нельзя держать вечно.

## Классы порчи

- **weak** — базовая
- **strong** — усиленная
- **severe** — редкая/ивентовая

## Правило перекрытия сильной порчи

Сильная (strong/severe) порча **не снимается одним артефактом**. Полное снятие:

- либо **2 артефакта «cleanse»**
- либо **Defense Tier ≥ 2**

## Defense Tier (уровень защиты)

Считается по защитным зданиям на поле:

- **Tier 1:** есть хотя бы 1 защитное здание
- **Tier 2:** Defense Power ≥ 14 → resist +8%, 1 cleanse снимает strong
- **Tier 3:** Defense Power ≥ 26 → resist +15%, 1 cleanse снимает severe (шанс 50%)

| 3 | 45 мин | 4 ч |
| 4 | 60 мин | 6 ч |
| 5 | 90 мин | 8 ч |

## Формула итоговой порчи

```
P_effect = P_base * (1 + attacker_power_bonus/100) * (1 - target_resist/100)
P_effect = clamp(P_effect, 0, cap_effect)  // cap 25%
```

## Defense Power (веса зданий)

| Здание | Weight |
|--------|--------|
| guard_post | 1.0 |
| watchtower | 1.2 |
| infirmary | 1.0 |
| temple | 1.4 |
| silence_chapel | 1.1 |
| treasury | 0.6 |
| well | 0.6 |
| firebrigade | 0.7 |

`def_power = sum(level * weight)` по защитным зданиям. Tier2: ≥14, Tier3: ≥26.

## Impact score (класс порчи)

```
impact_score = abs(effect_value_pct) + duration_hours * 1.5
```
- weak: < 20
- strong: 20..39
- severe: ≥ 40

## Суточные капы покрытия (бафы)

| Группа | Кап/сутки |
|--------|-----------|
| check-in | 6 ч |
| defense | 4 ч |
| curse | 3 ч |

Превышение → следующая активация даёт 50% длительности.
