# Экономика и пулы

## Распределение TON

- **30%** — проекту (treasury_project)
- **70%** — в призовой пул (treasury_prize)

```
project_fee = pay_ton * 0.30
pool_fee    = pay_ton * 0.70
```

## Касса (Treasury)

- **treasury_project** — 30% от TON-доходов  
- **treasury_prize** — 70%  
- **treasury_buffer** — страховка на выводы, например 15% от prize  
- **payout_pool** — откуда идут выплаты топам  

```
buffer_from_prize_pct = 15
payout_pool += (ton_in * 0.70) * 0.85
```

## Доход/час и оффлайн

```
income_per_hour_b = max(0, income_base_b * (1 + income_pct_bonus_b/100) + income_flat_bonus_b) * ads_multiplier
income_per_hour_total = sum(income_per_hour_b)
cap_offline_hours = 12
earned = floor(income_per_hour_total * min(dt_hours, cap_offline_hours))
```

## Доли пула наград (шахта / магазин)

В конфиге задаются доли общего фонда на эру: `rewardPoolShareMine` (%), `rewardPoolShareShop` (%). Шахта получает свою долю в виде весов дропа (в т.ч. токены STARS, печки); магазин — пополнение лимитированных предложений или фонд на награды. Таблицы — [18_Dev_таблицы_и_формулы.md](18_Dev_таблицы_и_формулы.md).

## «Проект в плюсе»

- Вывод TON только из **payout_pool** (и выделенного казначейства).
- Монеты — внутриигровые, не конвертируются в TON напрямую.
